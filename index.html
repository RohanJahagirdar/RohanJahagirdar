Linkem

Link'em is a simple and addictive game to play. The task is very simple; start connecting as many same-colored dots as you can in 60 seconds. Each time you create a closed loop of the same colored dots, all the remaining dots of the same color on the board are removed and added to your points tally. You can pause the game and revert back if needed.
